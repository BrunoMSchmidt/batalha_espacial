# Batalha Espacial
Batalha Espacial é um jogo inspirado no jogo batalha naval, onde a mecânica é basicamente a mesma, só que com naves espaciais ou foguetes no lugar de navios.

# Tecnologias Utilizadas
 - ReactJS
 - Typescript
 - ElectronJS
 - HTML
 - NodeJS
